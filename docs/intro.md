# Welcome to SRLINES Docs

SRLINES helps teams build WhatsApp automation, campaigns, and support workflows on top of the WhatsApp Cloud API.

## Why SRLINES

SRLINES combines onboarding, automation, integrations, analytics, and governance in one platform for revenue and support teams.

## Who this documentation is for

Product owners, developers, support managers, growth marketers, and founders operating WhatsApp-driven workflows.

## How to use these docs

Start with Getting Started, then move to Fundamentals, WhatsApp Cloud API, and Automation. Use Tutorials for production-ready playbooks.
